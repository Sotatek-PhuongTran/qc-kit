# Q&A

Nơi tổng hợp câu hỏi & trả lời về QC-kit. Bạn có thể:

- 📖 **Tra cứu** câu hỏi đã có trong [FAQ & Troubleshooting](faq.md)
- ✍️ **Submit câu hỏi mới** nếu chưa có trong FAQ

---

## 📖 Đã có trong FAQ?

Trước khi submit, hãy kiểm tra trang [FAQ & Troubleshooting](faq.md) — gồm 26 câu hỏi thường gặp về:

- Setup & Cấu hình (path, onboarding, trigger, permission, API key, network)
- Output & Template (format sai, fabricate, ngôn ngữ, số lượng test case)
- File & Versioning (overwrite, conflict, archive)
- Performance (chậm, tốn token)
- Bảo mật (API key bị lộ, password trong config)
- Collaboration (BA chậm trả lời, re-audit)
- Khác (skip skill, NFR, mobile, customize template)

## 📋 Quy ước submit câu hỏi

Để câu hỏi được trả lời nhanh, vui lòng cung cấp:

- **Tool đang dùng**: Claude Code / Antigravity / Codex CLI
- **Skill đang chạy**: tên skill cụ thể
- **Lỗi / triệu chứng**: copy nguyên đoạn lỗi nếu có
- **Đã thử**: các bước bạn đã làm để xử lý
- **Attach** screenshot / log nếu giúp ích

Câu hỏi rõ ràng → trả lời nhanh hơn 5-10 lần.
