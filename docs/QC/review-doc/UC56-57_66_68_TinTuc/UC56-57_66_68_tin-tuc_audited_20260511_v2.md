## 🟢 What's Good (Sau re-audit)

1. **Conflict CMR-15 vs UC66 đã giải quyết triệt để:** UC v1.2 áp dụng CMR-15 rõ ràng — cho phép chọn 1 ngày, không chọn ngày nào thì đóng modal bình thường. Không còn mâu thuẫn.
2. **Edge cases ảnh null/lỗi được mô tả hoàn chỉnh:** Thumbnail null, ảnh cover null, ảnh inline lỗi — tất cả đều hiển thị placeholder icon. Wireframe mới xác nhận visual.
3. **Partial API failure có rule rõ ràng:** Section thành công hiển thị bình thường, section lỗi báo riêng — không block toàn bộ. Đây là improvement quan trọng cho testability.
4. **Carousel behavior chi tiết:** 1 bài = tĩnh, ≥2 bài = auto-scroll 5s + cuộn ngang, không dot indicator. Đủ thông tin để viết test case.
5. **CMR-18 Debounce áp dụng rõ ràng:** Tap card bài viết và đổi tab đều có debounce — ngăn double navigation.
6. **Đa ngôn ngữ UI labels hoàn chỉnh:** Tên 19 Tab, label "Tin tức mới nhất", placeholder "Tìm kiếm tin tức..." đều được dịch theo CMR-17.
7. **Lazy load mất mạng:** Giữ data cũ + hiển thị lỗi — UX tốt, testable.
8. **Android Back trên modal:** Quay lại màn trước — rõ ràng cho Android testing.
9. **Tên người đăng dài:** Truncate với `...` khi vượt max-width — consistent với Title/Excerpt truncate rule.
10. **AC4 cập nhật chính xác:** Phản ánh đúng CMR-15 behavior mới.

## 🧪 Testability Outlook (Sau re-audit)

**Có thể bắt đầu test ngay — Toàn bộ scope:**

- **Màn Danh sách Tin tức:** Header cố định "Tin tức", 19 Tab (đa ngôn ngữ), carousel Tin nổi bật (1 bài = tĩnh, ≥2 = auto-scroll 5s, no dot, no lazy load, exclusion rule), list Tin mới nhất (lazy load 20/lần, sort mới nhất), tìm kiếm gần đúng theo Tab active (CMR-01, debounce 3s, 500 chars, no clear X icon, non-sticky), filter ngày (CMR-15, cho phép 1 ngày, không chọn = đóng modal), Pull to Refresh (CMR-13), Skeleton Loading, CMR-18 debounce.
- **Modal Filter:** Date Range Picker (CMR-15), "Nhập lại" (reset, popup giữ), "Áp dụng" (1 ngày OK, 0 ngày = đóng), đóng bằng X/tap ngoài, Android Back = quay lại màn trước.
- **Màn Chi tiết:** Ảnh bìa (placeholder nếu null — THAY ĐỔI từ v1.1), Tag, Tiêu đề, Meta (tên dài truncate), Rich Text (ảnh inline lỗi = placeholder, no zoom), section 5 bài mới nhất (lọc ngôn ngữ, không scope Tab, loại trừ bài hiện tại).
- **Edge cases ảnh:** Thumbnail null = placeholder icon, ảnh cover null = placeholder (không ẩn), ảnh inline lỗi = placeholder.
- **Partial failure:** Section thành công hiển thị, section lỗi báo riêng.
- **Lazy load mất mạng:** Giữ data cũ + lỗi CMR-07.
- **Quy tắc ngôn ngữ:** Lọc bài viết + dịch UI labels (Tab, label, placeholder) theo CMR-17.
- **Error states:** Lỗi mạng, HTTP 500, HTTP 404 (bài xóa từ CMS), offline, partial failure, empty state.

**Không block test nhưng cần lưu ý:**

- **Q13 (Active filter indicator):** CMR-02 không liệt kê UC56-68 → QA test theo UC mô tả (không có indicator). Nếu dev implement indicator → raise question.
- **Q17 (Skeleton vs Spinner đổi tab):** UC nói Skeleton → QA test Skeleton. Nếu thực tế là Spinner → raise bug hoặc clarification.
- **Q9 (Portrait):** Assume portrait-only. Nếu app hỗ trợ landscape → cần test thêm.
- **Q14 (Sanitize):** Security testing ngoài scope functional test. Ghi nhận cho security review.

**Suggested test focus areas:**

- **Happy path:** Sidebar → "Tin tức" → Tab "Kinh tế" → Tin nổi bật carousel → Chi tiết → Tin tức mới nhất cuối trang → Chi tiết khác → Back
- **Filter flow:** Mở filter → chọn 1 ngày → Áp dụng → kết quả lọc đúng; đổi tab → filter giữ; không chọn ngày → đóng modal
- **Search flow:** Tap icon → nhập keyword → debounce 3s → kết quả trong Tab active; đổi tab → keyword giữ; tap lại icon → đóng
- **Carousel:** 0 bài = ẩn section; 1 bài = tĩnh; ≥2 bài = auto-scroll 5s + cuộn ngang
- **Ảnh null/lỗi:** Thumbnail null, cover null, inline lỗi → tất cả hiển thị placeholder icon
- **Partial failure:** API Tin nổi bật OK + API Tin mới nhất lỗi → section 1 hiển thị, section 2 báo lỗi riêng
- **Lazy load mất mạng:** Cuộn xuống cuối → mất mạng → data cũ giữ + lỗi hiển thị
- **Debounce:** Double-tap card → chỉ navigate 1 lần; double-tap tab → chỉ load 1 lần
- **Đa ngôn ngữ:** Đổi ngôn ngữ → bài viết lọc lại + Tab names/labels dịch
- **Android Back:** Modal đang mở + Back = quay lại màn trước
- **Truncate/Wrap:** Tên dài truncate `...`, Tag dài wrap, Title/Excerpt 2 dòng + ellipsis
