# Coverage Notes

- 56 test cases total: 24 GUI + 32 FUNC.
- Out-of-scope items (per skill scope): performance load testing, security penetration (XSS / SQL injection payload exploration), reCAPTCHA bypass — flagged separately for security/perf test plan.
- Visibility-only test scope per Q15 applied to: Logo (TC_001), Page Title (TC_001, TC_022), Character Counter visibility/increment (TC_001, TC_004, TC_013, TC_045) — no boundary-color tests pending design.
- Test data conventions:
  - Valid Email (unique): `john@example.com`, `jane@techhub.com`
  - Existing Email (non-Rejected): `existing@example.com`
  - Existing Email (Rejected): `rejected@example.com`
  - Valid Phone (E.164 with Sri Lanka country code): `+94771234567`, `+94772345678`
  - Existing Store Name (non-Rejected): `Existing Store`
  - Existing Store Name (Rejected): `Rejected Store`
